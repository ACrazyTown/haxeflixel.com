crazysam
5 months 3 days ago

I solved the  flash.utils::ByteArray/_uncompress()  errors.

In Tiled, go to Edit->Preferences and make sure Store tile layer data as:  is set to Base64 (gzip compressed).

I was previously using zlib compress, changing it to gzip compression fixed my issues.
