How to Fade in a FlxText?

SoulBeaver, 3 months 1 week ago

I'm using flixel 3.0.0-dev from git.

I'm using FlxTween.multiVar to help me change the alpha of a FlxText. Fading out works beautifully, just as one would expect:

FlxTween.multiVar(_title, { alpha: 0.0 }, 2.0, null);

However, I cannot for the life of me do the reverse:

_title.alpha = 0.0; FlxTween.multiVar(_title, { alpha: 1.0 }, 2.0, null);

The FlxText stays completely transparent. What am I doing wrong?

Full source here:

override public function create():Void { FlxG.state.bgColor = 0xFF101414; _title = new FlxText(0, FlxG.height / 5, 500, "Turn-Based RPG"); _title.setFormat(null, 48, FlxColor.WHITE, "center", FlxText.SHADOW); _title.antialiasing = true; center(_title); add(_title); _title.alpha = 0.0; FlxTween.multiVar(_title, { alpha: 1.0 }, 2.0, null); super.create(); }

3 months 1 week ago

I believe to have found the problem. The current function FlxText.setFormat() function does not explicitly call calcFrame, instead setting _regen = true instead. However, an explicit call is required.

Original function:

public function setFormat(?Font:String, Size:Float = 8, Color:Int = 0xffffff, ?Alignment:String, BorderStyle:Int=NONE, BorderColor:Int=0x000000):FlxText { if (_isStatic) { return this; } if (Font == null) { _format.font = FlxAssets.FONT_DEFAULT; } else { _format.font = Assets.getFont(Font).fontName; } _format.size = Size; Color &= 0x00ffffff; _format.color = Color; _format.align = convertTextAlignmentFromString(Alignment); _textField.defaultTextFormat = _format; updateFormat(_format); borderStyle = BorderStyle; borderColor = BorderColor; _regen = true; return this; }

Modified function:

public function setFormat(?Font:String, Size:Float = 8, Color:Int = 0xffffff, ?Alignment:String, BorderStyle:Int=NONE, BorderColor:Int=0x000000):FlxText { if (_isStatic) { return this; } _format = new TextFormat(FlxAssets.FONT_DEFAULT, Size, Color); if (Font != null) { _format.font = Assets.getFont(Font).fontName; } _format.align = convertTextAlignmentFromString(Alignment); _textField.defaultTextFormat = _format; updateFormat(_format); borderStyle = BorderStyle; borderColor = BorderColor; #if flash calcFrame(); #else calcFrame(true); #end return this; }

If this change is correct, then I would like to commit it. However, I've never really committed to an OSS before, what is the standard procedure?
