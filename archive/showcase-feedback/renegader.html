Hey there,

I really like thsoe graphics! Anyone immediately notices that you're a "graphics person". :) That doesn't have to mean that you can't do programming, though! 

First off, when it comes to controls, always allow different options. I know some people that refuse to play any game with the arrow keys, because they're used to using their left hand for directional movement. It takes about 1 minute to implement WASD movement. Same for "Z to start", could as well be "any key". ;D

The game doesn't feel very good right now. Being taken back to the title screen immediately after you die feels very wrong. There need to be soundeffects, explosions and all that fun stuff.

Some soundeffects in general would be great. Check out

  • http://freesound.org/
  • http://www.bfxr.net/

The title screen is a little static right now. Could use some animations, like the title flying across the screen, or scaling up to its current size.. The possibilities are endless. Have a look at FlxTween!

You absolutely need to keep track of the "current best" in a game like this, and display in-game and preferably on the menu as well.

The score text could be a bit more readable. Maybe add a drop shadow to the text, or a slightly opaque, black FlxSprite.

You should probably hide the mouse, it's not needed in this game.

I took me way too long to find out that you can shoot with Z. Not that obvious, actually! Again, needs soundeffects. I initially thought this game was purely about avoiding. Destroyed enemies need to have particle explosions (via FlxEmitter!). Also, it's no fun that you have to repeatedly mash a key to shoot. That just annoys the player. What reason would the player have not to shoot at all times? Correct, there's absolutely none. So either limit this ability (seems a little OP anyway?) or make it an auto-fire.

How could this be "limited"? Well, for one, you could switch the gun with a rockte launcher. Again, giant explosions, those are fun. Ammo would be limited and you could pick it up every so often. Using your weapon you do as a last resort.

The game ends a bit too quickly for my taste right now. Either implement a life system (about 3 seems right?) or let the game start out a bit slower. Not sure if it ramps up, but it definitely starts too quick. Keep in mind, as the dev, you sense ot difficulty is totally off, having played the game over and over again.

While the graphics are good, there's not nearly enough variation and things quickly start to look very boring. Literally every single enemy uses the same color scheme, and there only seem to be two types of enemy vehicles anyway. Same for the background, which you were asking for specifically. Although it currently doesn't bother me quite as much, honestly.

On a technical level, tilemaps would allow for some very advanced backgrounds. FlxBackdrop not so much I guess.

Don't take the player back to the title screen once he dies. Do everything you can to make him play again! I'd just display some game over message and if any key is pressed (except for escape, which takes you back to the menu), a new game is started. Some games even go as far as restarting the game immediately without asking the player!

The game needs some music! Check out http://www.newgrounds.com/audio/ for some free, but awesome stuff.

Phew, I guess that's enough for now. Sorry for the wall of text, hope it helps! :)

This is a bit plain right now. Like, the gameplay after 10s and 5m still is the same. No room for improvement or learning, as there is no new things to master.

Moving the character feels a bit rough too. I guess you're doing something like 

if(right pressed) x += 2;

 

Try instead doing something like 

if(right pressed) speedHorizontal += acceleration; speedHorizontal *= friction; x += speedHorizontal;

 

I guess Flixel can do the smooth movement thing on its own, but I've never actually tried.

 

 

Also the graphics were nice. Having inspiration is a thing, we all have, but try to do your own thing instead of copying others, it will be more rewarding for sure :)

 

Good luck on this one, and keep us informed of your progress!

 

EDIT: Just noticed it, but the flashing effect when hitting an enemy is nice! This is the kind of feedback/polish you need to make a juicy game :D
