Pretty polished (especially the graphics, even though simplistic) and good game overall - considering it's your first! :) It's a good idea to start with a small project - everything else is doomed to fail. Me being me, I have a few complaints though:

  • The dimensions of the game seemed a tad too big. With so much space being between the two squares, I constantly had to move my eyes in order to compare the colors of them, which is after all the main objective. It would be much better if the dimensions were small enough for you to be able to see both squares at one glance.
  • Multiple options are always good, especially when it comes to key bindings. Tab to go back in the menu felt a little counter-intuitive. I feel like escape should work as well. Enter should work as an alternative for space. R to restart the game.
  • I didn't really like the GUI. It seems like the "records" submenu isn't really necessary. That information would fit in the main menu as well - and also should be there, at least the "best so far". My feelings about the "settings" submenu are similar, the ability to switch difficulty modes is hidden away too much. Instead, you should be asked what difficulty to play on after pressing play. The volume setting would be obsolete if you used the flixel soundtray, and the ability to toggle the tutorial on and off seems a little superfluous in a game as simple as this. The message that tells you that you can turn the tutorial message off is almost as long as the tutorial message itself! Instead, it should only appear when you play for the first time.
  • The gameplay music kind of ruined it for me. Not because it's bad, but because I've heard it so damn often. It's used - like everywhere! :/ Check out the NG audio portal sometime.
  • When it comes to gameplay - it should be more forgiving. Even easy mode seems really hard when you just start out and try to understand how the game works - partly because the timer is already running while you're still reading the tutorial.
  • Not sure I really like the controls with left and right to "browse through" the colors. There only seem to be three, so keys 1-3 should at least work as an alternative. Also, it'd be nice if there was an indicator somehwere on the screen that shows the colors available and which one you're on right now.

I hope you agree with at least some of these thoughts and that they'll help you to create (even) better games in the future! :)
