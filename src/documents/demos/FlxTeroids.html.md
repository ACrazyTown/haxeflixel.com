FlxTeroids is a simple Asteroids clone. The original GitHub repo by [Adam "Atomic" Saltsman](https://twitter.com/ADAMATOMIC) can be found [here](https://github.com/AdamAtomic/FlxTeroids).

**Controls:**

Move - Arrow keys

Shoot -Space
